## Contributing

1. Fork it
1. Get it running
1. Create your feature branch (`git checkout -b my-new-feature`)
1. Write your code and **specs**
1. Run Rubocop (`bundle exec rubocop`), fix any warnings
1. Commit your changes (`git commit -am 'Add some feature'`)
